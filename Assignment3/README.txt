This project allows users to convert units for temperature, length, and time. Simply select the measurement type, unit you are converting from
unit you are converting to, and enter the value of the unit. Press calculate to get the converted value rounded to 2 decimal places.
Requirements:
	Python installed
	tkinter installed (can check if it is installed with: python -m tkinter
Double click 'UnitConversionTool.py' to open interface
Follow instructions on the interface to convert units.
